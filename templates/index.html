Page templates

Four blank starting points for Gloucester City Council web pages. Each is fully accessible, includes the breadcrumb, and uses the same brand styling. Open one to preview, then duplicate the file to start a new page.

The floating Accessibility button (bottom right) gives every template controls for text size, contrast, dark theme, reading aids, motion, and the new Reading ruler.

①Template 1 ·Service landing

Service landing page

For the top of a service area. Hero, service-card grid, popular tasks, contact strip.

Open template Docs
②Template 2 ·Transaction

Transaction page

For "do something" pages. Includes a Start-now CTA pattern and an inline form with error summary, inline errors and a success state.

Open template Docs
③Template 3 ·Information

Information page

For explaining a topic. Two columns, on-page tab nav, step list, table, definition list, inset panels, sidebar.

④Template 4 ·FAQ

FAQ page

Categorised questions using native <details>/<summary> accordions — works without JavaScript.
