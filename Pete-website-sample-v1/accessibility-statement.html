Accessibility statement

This accessibility statement applies to the Council Services website.

This website is run by District Council. We want as many people as possible to be able to use this website. For example, that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We have also made the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible this website is

We know some parts of this website are not fully accessible:

  • This is a demonstration website and some interactive elements are not fully functional
  • Some service links are placeholder links for demonstration purposes

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the 'accessibility regulations'). If you are not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about this website's accessibility

District Council is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Compliance status

This website is partially compliant with the Web Content Accessibility Guidelines version 2.2 AA standard, due to the non-compliances listed below.

Non-compliance with the accessibility regulations

This is a demonstration website and some interactive elements are not fully functional. These would be fully compliant in a production environment:

  • Search functionality demonstrates user interface only
  • Service card links are placeholder demonstrations

What we are doing to improve accessibility

We are committed to ensuring this website meets WCAG 2.2 AA standards. We:

  • regularly review and test the website for accessibility issues
  • provide training to content editors on accessibility best practices
  • follow GOV.UK design patterns and accessibility guidelines
  • test with assistive technologies including screen readers
  • ensure all new content meets accessibility standards

Preparation of this accessibility statement

This statement was prepared on 13 February 2026. It was last reviewed on 13 February 2026.

This website was last tested on 13 February 2026. The test was carried out internally following WCAG 2.2 AA guidelines.

Accessibility features

This website includes the following accessibility features:

  • High contrast mode toggle
  • Skip to main content link for keyboard users
  • Semantic HTML structure with proper heading hierarchy
  • ARIA labels and landmarks for screen readers
  • Keyboard navigation support throughout
  • Focus indicators meeting WCAG 2.2 requirements
  • Colour contrast ratios meeting AA standards
  • Responsive design supporting text resizing up to 200%
  • Plain language with maximum reading age of 11
  • Alternative text for all meaningful images
  • Consistent navigation and layout
