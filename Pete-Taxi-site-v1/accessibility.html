Accessibility statement

This accessibility statement applies to the Gloucester City Council hackney carriage and private hire licensing website.

Our commitment to accessibility

Gloucester City Council is committed to making this website accessible in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This website aims to meet the Web Content Accessibility Guidelines (WCAG) 2.2 AA standard as a minimum.

Compliance status

This website is partially compliant with WCAG 2.2 AA. We are actively working to identify and resolve any issues that prevent full compliance.

How we make this website accessible

  • All pages have a skip to main content link to help keyboard users bypass repeated navigation
  • All images have meaningful alternative text, or are marked as decorative
  • Colour is not the only way to convey meaning — text labels and icons are used alongside colour
  • All interactive elements can be used with a keyboard alone
  • Focus states are clearly visible and meet WCAG contrast requirements
  • Form fields have clear, visible labels
  • The site respects user preferences for reduced motion
  • A high contrast mode is available using the button in the header
  • Content is written in plain English with a reading age of no more than 11 years
  • All text uses UK English
  • Heading levels are used in the correct order on each page
  • Each page has a single H1 heading
  • Tables have captions and appropriate header cells
  • Pages are tested using automated tools and screen readers

Technical information

This website is built using HTML, CSS and JavaScript. It has been tested with the following browsers and assistive technologies.

  • Google Chrome (latest) on Windows and macOS
  • Mozilla Firefox (latest) on Windows and macOS
  • Apple Safari (latest) on macOS and iOS
  • Microsoft Edge (latest) on Windows
  • NVDA screen reader on Windows
  • VoiceOver on macOS and iOS
  • Keyboard-only navigation

Known accessibility issues

We are not currently aware of any significant accessibility issues. If you find a problem, please tell us using the contact details below.

Requesting information in a different format

If you need the information on this website in a different format — such as large print, audio or braille — contact the licensing team.

We will consider your request and respond within 15 working days.

Reporting an accessibility problem

If you experience an accessibility barrier on this website, please tell us by emailing licensing@gloucester.gov.uk or calling 01452 396396.

Please include as much detail as possible, including the page you were on and what you were trying to do.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the accessibility regulations. If you are not happy with how we respond to your accessibility complaint, contact the Equality Advisory and Support Service (EASS).

Preparation of this statement

This statement was prepared in June 2025. It was last reviewed in February 2026.

The website was tested by Gloucester City Council's digital team using automated accessibility tools and manual testing with assistive technology.
