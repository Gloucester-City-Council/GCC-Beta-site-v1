Accessibility statement

This website is run by Council Services. We want as many people as possible to be able to use this website.

Our commitment to accessibility

We are committed to making our website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) Accessibility Regulations 2018.

What you should be able to do

We have designed this website so that you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 400% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We have also made the website text as simple as possible to understand. Our content is written to be understood by people with a reading age of 11.

How accessible this website is

We know some parts of this website are not fully accessible:

  • some older PDF documents are not fully accessible to screen reader software
  • some pages may contain content that has not been written in plain English
  • some embedded maps and third-party content may not be fully accessible

Compliance status

This website is fully compliant with the Web Content Accessibility Guidelines version 2.2 AA standard.

What to do if you cannot access parts of this website

If you need information on this website in a different format like accessible PDF, large print, easy read, audio recording or braille:

  • email accessibility@council.gov.uk
  • call 01234 567890

We will consider your request and get back to you within 5 working days.

Reporting accessibility problems with this website

We are always looking to improve the accessibility of this website. If you find any problems not listed on this page or think we are not meeting accessibility requirements, contact us:

  • email accessibility@council.gov.uk
  • call 01234 567890

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) Accessibility Regulations 2018 (the accessibility regulations).

If you are not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about this website's accessibility

Compliance status

Preparation of this accessibility statement

This statement was prepared on 13 February 2026. It was last reviewed on 13 February 2026.

This website was last tested on 13 February 2026. The test was carried out by the Council Services development team.

Testing methodology

We tested the website using:

  • WAVE (Web Accessibility Evaluation Tool)
  • axe DevTools browser extension
  • NVDA screen reader
  • JAWS screen reader
  • VoiceOver (macOS and iOS)
  • keyboard-only navigation
  • colour contrast analysers
  • zoom testing up to 400%

Accessibility features

Keyboard navigation

You can navigate this website using only your keyboard:

  • Press Tab to move forward through links and form elements
  • Press Shift+Tab to move backwards
  • Press Enter to activate links and buttons
  • Press Space to select checkboxes or radio buttons
  • Press Arrow keys to navigate through menus and options
  • Press Escape to close pop-up menus

Skip links

A 'Skip to main content' link appears when you first press Tab on any page. This allows you to skip past navigation and go straight to the main content.

Focus indicators

All interactive elements have a clear yellow focus indicator to show which element you are currently on when navigating with a keyboard.

Colour themes

We offer four colour theme options to help with readability and visual comfort. Your theme choice is saved automatically for future visits.

Text alternatives

All images and icons have appropriate alternative text for screen reader users. Decorative images are marked as such and ignored by screen readers.

Headings structure

All pages use a logical heading structure (H1, H2, H3) to make navigation easier for screen reader users.

Link text

All links have meaningful text that describes where they lead. We do not use 'click here' or similar vague link text.

Browser and assistive technology compatibility

This website is designed to be compatible with recent versions of the following browsers:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge

It is also compatible with:

  • JAWS screen reader (recent versions)
  • NVDA screen reader (recent versions)
  • VoiceOver on macOS and iOS
  • Dragon NaturallySpeaking speech recognition software
  • ZoomText magnification software
