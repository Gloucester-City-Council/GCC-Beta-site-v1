# Logo Image

Please place the Gloucester City Council logo file in this directory with the filename: `gcc-logo.png`

## Requirements

- **Filename**: `gcc-logo.png`
- **Recommended dimensions**: Width should be appropriate for 60px height (auto-width)
- **Format**: PNG (with transparent background recommended)
- **Alternative formats**: JPG, SVG also supported

## How to Add Your Logo

1. Save your Gloucester City Council logo as `gcc-logo.png`
2. Copy it to this directory: `Pete-website-sample-v2/assets/images/`
3. The logo will automatically appear in the header on all pages

## Notes

- The logo height is set to 60px on desktop and 50px on mobile
- The logo is clickable and links back to the homepage from other pages
- The alt text is set to "Gloucester City Council" for accessibility
- If using a different image format, update the src in the HTML files:
  - `index.html` line 17
  - `pages/licensing.html` line 18
  - `pages/accessibility.html` line 18
